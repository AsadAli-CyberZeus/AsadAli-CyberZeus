# GitHub Profile README Setup

## 1. Create the profile repository
Create a GitHub repository with the exact same name as your username:

```text
AsadAli-CyberZeus
```

The final path should be:

```text
https://github.com/AsadAli-CyberZeus/AsadAli-CyberZeus
```

## 2. Upload this package
Upload these files/folders into the repository root:

```text
README.md
assets/
.github/workflows/snake.yml
.github/workflows/profile-3d.yml
```

## 3. Preview before final commit
On GitHub, open `README.md`, paste the content, and click the **Preview** tab before committing.

For local preview, open `PREVIEW_STANDALONE.html` in your browser.

## 4. Run animations that need GitHub Actions
After pushing:

1. Go to the **Actions** tab.
2. Run `Generate contribution snake` manually.
3. Run `Generate 3D contribution profile` manually.
4. Wait 1–3 minutes.
5. Refresh your GitHub profile.

## Important
Typing animation, badges, stats, trophies, activity graph, and local animated SVG banners should show quickly.
Snake and 3D contribution graph only show after workflows generate their files.
